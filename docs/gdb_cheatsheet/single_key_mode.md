## **SINGLE KEY MODE**

| Shortcut           | Result                     |
| :----------------- | :------------------------- |
| `CTRL` + `X` + `S` | Switch to single key mode. |
| `q`                | Exit the SingleKey mode.   |
| `c`                | continue                   |
| `d`                | down                       |
| `f`                | finish                     |
| `n`                | next                       |
| `r`                | run                        |
| `s`                | step                       |
| `u`                | up                         |
| `v`                | info locals                |
| `w`                | where                      |
